# RotasEmergenciaPi
Projeto para Projeto Integrador

# Back-end
Spring boot com Java 17 usando maven com as dependencias H2, Jpa, Dev Tools e Spring Web

# Front-end 
Angular CLI versão 22 usando Node.js com Typesript(HTML,CSS) para uma aplicação Spa(Single page application)

# Documentaçao 
Em forma de PDF com atributos de Sistema Web usando Aws como opção Cloud, Exemplos do codigo como ER(Expressão Regular), Automatos Finitos, REGEX e GLC(Simulando um analisador léxico e sintático).
Tambem usamos Cometarios no codigo para exemplificar Padrões de projetos usados
